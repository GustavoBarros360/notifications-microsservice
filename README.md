## Description

This is a microsservice for manage notifications from a system. It was developed during the ignite-lab by rocketseat.

The tecnologies used here is the following:

- Prisma as the ORM
- SQLite
- Typescript
- NEST.JS

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
